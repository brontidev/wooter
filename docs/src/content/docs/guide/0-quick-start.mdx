# hello world!
Define routes using `.route()`

```ts
import { c, Wooter } from "@bronti/wooter"
const wooter = new Wooter()

wooter.route(c.chemin(), "GET", (ctx) => {
    ctx.resp("Hello world!")
})
```

# serving
Wooter does not implement an HTTP server, instead it exposes a single entry point: `.fetch(request: Request): Promise<Response>`

## using `export default`
In some environments, serving is as easy as exporting a default object with a fetch method:
```ts
export default wooter;
// or
const { fetch } = wooter;
export default { fetch };
```

This works out of the box for Cloudflare, Vercel, Bun, and Deno (using `deno serve`), and Deno Deploy.
